# 🕋 Hajj & Umrah Crowd Density Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting crowd density levels during Hajj and Umrah using machine learning techniques. It analyzes multiple factors such as movement patterns, environmental conditions, and crowd behavior to enhance safety and crowd management.

---

## 🎯 Objectives
- Predict crowd density levels accurately  
- Improve crowd control and management  
- Support decision-making during large-scale events  

---

## 🧠 Machine Learning Workflow
1. Data preprocessing and cleaning  
2. Feature engineering  
3. Model training  
4. Model evaluation  
5. Prediction  

---

## ⚙️ Algorithms Used
- Random Forest  
- Logistic Regression  
- Decision Tree  

---

## 📊 Results
The model demonstrates strong performance in predicting crowd density and can be applied in real-world scenarios such as crowd monitoring and safety planning.
